[![Build Status](https://travis-ci.com/redsalmon91/FoxSEE.svg?branch=master)](https://travis-ci.com/redsalmon91/FoxSEE)

## Features

- 0x88 Board
- Minimax Search with Alpha-Beta Pruning
- Quiescence Search
- History Heuristic
- Refutation Table
- Zobrist Hashing
- Bitboards
- Check Extension
- SEE
- MVV/LVA

## How to build
Run `cargo build --release`
